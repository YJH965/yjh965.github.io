黎明前的巧克力 / CF1906K Deck-Building Game

注意到原题方案数相当于选择一个异或和为 000 的子集 SSS 并乘上 2∣S∣2^{|S|}2∣S∣ 的系数。那么可以列出答案柿子为：

[x0]∏i=1n(1+2xai)[x^0] \prod_{i = 1}^n (1 + 2x^{a_i}) [x0]i=1∏n​(1+2xai​)

这可以使用类似分治 NTT 的东西来做到 O(n+m22m)\mathcal{O}(n + m^22^m)O(n+m22m)，具体来讲就是分别只保留最高位为 000 或 111 的部分递归下去，然后使用这两部分的结果分别合并出最后异或出来时最高位为 000 或 111 的结果。这样做时间复杂度正确的原理在于，如果设值域为 2m2^m2m，且确定了长度为 kkk 的前缀，那么这些东西只能异或出 O(2m−k)\mathcal{O}(2^{m - k})O(2m−k) 种结果。

但是我们也许可以做到更优。考虑求出这个东西 FWT 之后的结果，再 IFWT 回去：

我们设 smj,s=⨁p∈saj,psm_{j, s} = \bigoplus_{p \in s} a_{j, p}smj,s​=⨁p∈s​aj,p​，接着定义 hs=[xi]FWT⁡(∑j=1nxsmj,s)h_s = [x^i] \operatorname{FWT}\left(\sum_{j = 1}^n x^{sm_{j, s}}\right)hs​=[xi]FWT(∑j=1n​xsmj,s​)，则有：

ABC367G Sum of (XOR^K or 0)

我们直接考虑对于所有异或和求出对应序列的方案数，忽略这个 KKK 次方。

这个题其实就是上一个题的 k=2k = 2k=2 的情况（w1=1,w2=y,ai,1=0,ai,2=Aiw_1 = 1, w_2 = y, a_{i, 1} = 0, a_{i, 2} = A_iw1​=1,w2​=y,ai,1​=0,ai,2​=Ai​）。若我们直接套用上一题的做法，求 sm,h,csm, h, csm,h,c 都不涉及到 yyy，于是时间复杂度不变；预处理 cosco_scos​ 的次方时需要记录 yyy 相关的完整系数，每次需要乘上一个单项式，在本来 O(n2k)\mathcal{O}(n2^k)O(n2k) 的基础上多乘了一个 MMM；求 ∏s=02k−1coscs\prod_{s = 0}^{2^k - 1} {co_s}^{c_s}∏s=02k−1​cos​cs​ 时涉及到了循环多项式的卷积，在本来 O(n2k)\mathcal{O}(n2^k)O(n2k) 的基础上多乘了两个 MMM；最后 IFWT 回去的时候，只涉及到了 yyy 相关的多项式的数乘与加减操作，所以只记录 y0y^0y0 的系数即可，时间复杂度不变。综上所述，总时间复杂度为 O(nM22k+mlog⁡K+(k+m)2k+m)\mathcal{O}(nM^22^k + m \log K + (k + m)2^{k+m})O(nM22k+mlogK+(k+m)2k+m)（如果你闲着没事干可以线性筛预处理 xKx^KxK 来去掉 log⁡\loglog）。

CF2096H Wonderful XOR Problem

在这题里，我们要求的是：

[xk]FWT⁡(∏i=1n∑j=lirixj)=∏i=1n[xk]FWT⁡(∑j=1rixj−∑j=1li−1xj)=∏i=1n([xk]FWT⁡(∑j=1rixj)−[xk]FWT⁡(∑j=1li−1xj))\begin{aligned} &[x^k] \operatorname{FWT}\left(\prod_{i = 1}^n \sum_{j = l_i}^{r_i} x^j\right)\\ =&\prod_{i = 1}^n [x^k] \operatorname{FWT}\left(\sum_{j = 1}^{r_i} x^j - \sum_{j = 1}^{l_i - 1} x^j\right)\\ =&\prod_{i = 1}^n \left([x^k] \operatorname{FWT}\left(\sum_{j = 1}^{r_i} x^j\right) - [x^k] \operatorname{FWT}\left(\sum_{j = 1}^{l_i - 1} x^j\right)\right)\\ \end{aligned} ==​[xk]FWT⎝⎛​i=1∏n​j=li​∑ri​​xj⎠⎞​i=1∏n​[xk]FWT(j=1∑ri​​xj−j=1∑li​−1​xj)i=1∏n​([xk]FWT(j=1∑ri​​xj)−[xk]FWT(j=1∑li​−1​xj))​

我们设 f(r,i)f(r, i)f(r,i) 为 [xi]FWT⁡(∑j=1rxj)[x^i] \operatorname{FWT}\left(\sum_{j = 1}^r x^j\right)[xi]FWT(∑j=1r​xj)，那么我们要求的东西就变成了 ∏i=1n(f(ri,k)−f(li−1,k))\prod_{i = 1}^n (f(r_i, k) - f(l_i - 1, k))∏i=1n​(f(ri​,k)−f(li​−1,k))。考虑这个 f(r,i)f(r, i)f(r,i) 的性质：

f(r,i+2j+1)=f(r,i)f(2k−1,i)=2k[i mod 2k=0]f(r,i)={f(2j−1,i)+f(r−2j,i)i<2jf(2j−1,i)−f(r−2j,i)i≥2j (0≤i<2j+1)f(r,i)={r+1i=0f(r−2j,i)0<i<2j2j+1−r−1i=2j−f(r−2j,i)2j<i<2j+1 (0≤i<2j+1)f(r, i + 2^{j + 1}) = f(r, i)\\ f(2^k - 1, i) = 2^k[i \bmod 2^k = 0]\\ f(r, i) = \begin{cases} f(2^j - 1, i) + f(r - 2^j, i) & i < 2^j\\ f(2^j - 1, i) - f(r - 2^j, i) & i \ge 2^j\\ \end{cases} \ (0 \le i < 2^{j + 1})\\ f(r, i) = \begin{cases} r + 1 & i = 0\\ f(r - 2^j, i) & 0 < i < 2^j\\ 2^{j + 1} - r - 1 & i = 2^j\\ -f(r - 2^j, i) & 2^j < i < 2^{j+1}\\ \end{cases} \ (0 \le i < 2^{j + 1})\\ f(r,i+2j+1)=f(r,i)f(2k−1,i)=2k[imod2k=0]f(r,i)={f(2j−1,i)+f(r−2j,i)f(2j−1,i)−f(r−2j,i)​i<2ji≥2j​ (0≤i<2j+1)f(r,i)=⎩⎪⎪⎪⎨⎪⎪⎪⎧​r+1f(r−2j,i)2j+1−r−1−f(r−2j,i)​i=00<i<2ji=2j2j<i<2j+1​ (0≤i<2j+1)

如果我们从高到低按位考虑，我们可以写出计算 f(r,i)f(r, i)f(r,i) 的代码：

inline int calc(int r, int i, int p = n - 1){
	if (!i) return r + 1;
	if (r >> p & 1 ^ 1){
		if (i >> p & 1) i ^= 1 << p;
		return calc(r, i, p - 1);
	}
	if (i == 1 << p) return (1 << p + 1) - r - 1;
	return i >> p & 1 ? -calc(r ^ 1 << p, i ^ 1 << p, p - 1) : calc(r ^ 1 << p, i, p - 1);
}

稍微进行一些观察，我们发现对于一个固定的 rrr，f(r,i)f(r, i)f(r,i) 的绝对值只与 iii 的 lowbit 有关，而正负性是好求的，可以做到 O(1)\mathcal{O}(1)O(1)。

inline int calc(int r, int i){if (!i) return r + 1;
	int p = __builtin_ctz(i), neg = popc((r & i) >> p + 1) & 1 ? -1 : 1;

考虑枚举 kkk 的 lowbit 为 ppp，即 kkk 的第 ppp 位为 111，更低位全为 000，更高位任意，将 lowbit 相同的 kkk 放在一起算。

枚举 ppp 后有 f(ri,k)=ai(−1)∣k′∩ri′∣f(r_i, k) = a_i(-1)^{|k' \cap r'_i|}f(ri​,k)=ai​(−1)∣k′∩ri′​∣，其中 aia_iai​ 只与 rir_iri​ 有关，k′=k/2p+1,ri′=ri/2p+1k' = k / 2^{p + 1}, r'_i = r_i / 2^{p + 1}k′=k/2p+1,ri′​=ri​/2p+1，f(li−1,k)f(l_i - 1, k)f(li​−1,k) 也是同理。那么我们所求的 ∏i=1n(f(ri,k)−f(li−1,k))\prod_{i = 1}^n (f(r_i, k) - f(l_i - 1, k))∏i=1n​(f(ri​,k)−f(li​−1,k)) 就转化成了 ∏i=1n(ai(−1)∣k′∩ci∣+bi(−1)∣k′∩di∣)\prod_{i = 1}^n (a_i(-1)^{|k' \cap c_i|} + b_i(-1)^{|k' \cap d_i|})∏i=1n​(ai​(−1)∣k′∩ci​∣+bi​(−1)∣k′∩di​∣) 的形式，而 ai,bi,ci,dia_i, b_i, c_i, d_iai​,bi​,ci​,di​ 都是可以事先 O(1)\mathcal{O}(1)O(1) 求出来的。

∏i=1n(ai(−1)∣k′∩ci∣+bi(−1)∣k′∩di∣)=∏i=1n[xk′]FWT⁡(aixci+bixdi)=[xk′]FWT⁡(∏i=1n(aixci+bixdi))=[xk′]FWT⁡(∏i=1nxci(ai+bixci⊕di))=[xk′]FWT⁡(∏i=1n(0+1xci)(ai+bixci⊕di))\begin{aligned} &\prod_{i = 1}^n (a_i(-1)^{|k' \cap c_i|} + b_i(-1)^{|k' \cap d_i|})\\ =&\prod_{i = 1}^n [x^{k'}] \operatorname{FWT}(a_ix^{c_i} + b_ix^{d_i})\\ =&[x^{k'}] \operatorname{FWT}\left(\prod_{i = 1}^n (a_ix^{c_i} + b_ix^{d_i})\right)\\ =&[x^{k'}] \operatorname{FWT}\left(\prod_{i = 1}^n x^{c_i}(a_i + b_ix^{c_i \oplus d_i})\right)\\ =&[x^{k'}] \operatorname{FWT}\left(\prod_{i = 1}^n (0 + 1x^{c_i})(a_i + b_ix^{c_i \oplus d_i})\right)\\ \end{aligned} ====​i=1∏n​(ai​(−1)∣k′∩ci​∣+bi​(−1)∣k′∩di​∣)i=1∏n​[xk′]FWT(ai​xci​+bi​xdi​)[xk′]FWT(i=1∏n​(ai​xci​+bi​xdi​))[xk′]FWT(i=1∏n​xci​(ai​+bi​xci​⊕di​))[xk′]FWT(i=1∏n​(0+1xci​)(ai​+bi​xci​⊕di​))​
