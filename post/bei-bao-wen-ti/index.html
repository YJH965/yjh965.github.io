多重背包

nnn 种物品，第 iii 种重量为 wiw_iwi​，价值为 viv_ivi​，有 cic_ici​ 个。有一个容量为 mmm 的背包，让你选择一些物品放进背包，在物品重量和 ≤m\le m≤m 的情况下使物品价值和最大。

最直接的方法是将每一种物品拆成 cic_ici​ 个一样的物品然后使用 01 背包，时间复杂度为 O(m×∑ci)O(m \times \sum c_i)O(m×∑ci​)。

对于每一种物品，设 ppp 为满足 ∑j=0p2j≤ci\sum_{j = 0}^p 2^j \le c_i∑j=0p​2j≤ci​ 的最大值，lll 为 ci−∑j=0p2jc_i - \sum_{j = 0}^p 2^jci​−∑j=0p​2j，我们可以将物品拆分为 (vi×20,wi×20),(vi×21,wi×21),⋯ ,(vi×2p,wi×2p),(vi×l,wi×l)(v_i \times 2^0, w_i \times 2^0), (v_i \times 2^1, w_i \times 2^1), \cdots, (v_i \times 2^p, w_i \times 2^p), (v_i \times l, w_i \times l)(vi​×20,wi​×20),(vi​×21,wi​×21),⋯,(vi​×2p,wi​×2p),(vi​×l,wi​×l)。

单调队列优化

不优化的转移是这样的：

fj=max⁡k=1ci{fj−k×vi+k×wi}f_j = \max_{k = 1}^{c_i}\{f_{j - k \times v_i} + k \times w_i\} fj​=k=1maxci​​{fj−k×vi​​+k×wi​}

这个式子很难优化，我们发现 fjf_jfj​ 总是会从 fj−k×vif_{j - k \times v_i}fj−k×vi​​ 转移，所以我们可以将 jjj 按照 mod  vi\mod v_imodvi​ 的余数分组，因为不同组之间不会互相影响，所以可以对每一组分别 DP。j∈[0,vi)j \in [0, v_i)j∈[0,vi​)（代表余数），转移方程变成这样：

fj+k×vi=max⁡l=k−cil−1{fj+l×vi+(k−l)×wi}f_{j + k \times v_i} = \max_{l = k - c_i}^{l - 1}\{f_{j + l \times v_i} + (k - l) \times w_i\} fj+k×vi​​=l=k−ci​maxl−1​{fj+l×vi​​+(k−l)×wi​}

这样子就可以使用单调队列优化了，时间复杂度为 O(nm)O(nm)O(nm)。

注意，余数 jjj 的循环顺序没有影响（每一组不会互相转移），但是 kkk 必须要倒着循环（原理同 01 背包）。

具体代码实现在这里：

#include <algorithm>
#include <cstdio>

using namespace std;

int n, m, v[1001], w[1001], c[1001], f[20001], ans;
int s, t, q[20001];
int main(){
	scanf("%d%d", &n, &m);
	for (int i = 1;i <= n;i ++) scanf("%d%d%d", &v[i], &w[i], &c[i]), c[i] = min(c[i], m / v[i]);
	for (int i = 1;i <= n;i ++){
		for (int j = 0;j < v[i];j ++){ // 枚举余数 j
			#define calc(x) (f[j + (x) * v[i]] - (x) * w[i])
			s = 1, t = 0;
			int p = (m - j) / v[i]; // 最大能选取的 k
			for (int k = p - 1;k >= max(p - c[i], 0);k --){
				while (s <= t && calc(q[t]) <= calc(k)) t --;
				q[++ t] = k;
			} // 先将初始决策插入单调队列（k = p 时需要的决策）
			for (int k = p;k >= 0;k --){
				while (s <= t && q[s] > k - 1) s ++; // 删除不合法的决策
				if (s <= t) f[j + k * v[i]] = max(f[j + k * v[i]], calc(q[s]) + w[i] * k); // 转移
				if (k - c[i] - 1 < 0) continue;
				while (s <= t && calc(q[t]) <= calc(k - c[i] - 1)) t --; // 维护单调性
				q[++ t] = k - c[i] - 1; // 插入新决策
			}
		}
	}
	for (int j = 0;j <= m;j ++) ans = max(ans, f[j]);
	printf("%d", ans);
}
