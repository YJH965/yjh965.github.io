树上差分学习笔记

2023-04-16 学习笔记

顾名思义，树上差分就是一个把差分搬到树上的 trick，可以解决一些树上的链问题。

前置知识：前缀和，LCA，树的存储与遍历。

树上差分

对询问进行差分

询问可以被差分的前提是，这个询问具有可减性，即具有逆运算。

比如说，询问链和是可以使用树上差分的，因为加法存在逆运算。

但是，询问链上 min⁡\minmin 值是不能使用树上差分的，因为 min⁡\minmin 运算没有逆运算。

设 dud_udu​ 为从根节点到 uuu 的答案，lcalcalca 为 uuu 和 vvv 的最近公共祖先，query⁡(u,v)\operatorname{query}(u, v)query(u,v) 为 (u,v)(u, v)(u,v) 链的答案，对询问进行差分的公式是 query⁡(u,v)=du+dv−dlca−dfalca\operatorname{query}(u, v) = d_u + d_v - d_{lca} - d_{fa_{lca}}query(u,v)=du​+dv​−dlca​−dfalca​​。

注意，上述公式针对信息在点上的情况，所以要保证 lcalcalca 的答案只被减掉一次，信息在边上的情况不需要保证，公式为 query⁡(u,v)=du+dv−2dlca\operatorname{query}(u, v) = d_u + d_v - 2d_{lca}query(u,v)=du​+dv​−2dlca​。

对修改进行差分

首先，我们回顾对序列区间修改进行差分的过程。

  • 给你一个序列 aaa，每次对 [l,r][l, r][l,r] 中的每个元素加 xxx，最后求出 aaa 的每个元素。

然后，我们把它放到树上。

  • 给你一棵树，节点 iii 有一个权值 aia_iai​，每次对 (u,v)(u, v)(u,v) 链的每个节点加 xxx，最后求出这棵树的每个节点的权值。

还是考虑维护 aaa 的差分 bbb，可是这次不同，bi=ai−∑j∈son⁡(i)ajb_i = a_i - \sum_{j \in \operatorname{son}(i)} a_jbi​=ai​−∑j∈son(i)​aj​。然后，aia_iai​ 为 iii 子树内的 bib_ibi​ 之和。类似的，设，lcalcalca 为 uuu 和 vvv 的最近公共祖先，每次修改，只有 bu=bu+1b_u = b_u + 1bu​=bu​+1，bv=bv+1b_v = b_v + 1bv​=bv​+1，blca=blca−1b_{lca} = b_{lca} - 1blca​=blca​−1，bfalca=bfalca−1b_{fa_{lca}} = b_{fa_{lca}} - 1bfalca​​=bfalca​​−1。最后对 bbb 做子树和和即可还原 aaa 序列。
