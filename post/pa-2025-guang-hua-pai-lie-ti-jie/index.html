先考虑第一问的答案是什么。考虑对于一个长为 nnn 的排列，我们对于每个位置 iii 求出 fif_ifi​ 表示以 iii 结尾的 LIS 长度，gig_igi​ 为以 iii 开头的 LDS 长度。那么我们可以定义一个二元组序列 aaa，其中 ai=(fi,gi)a_i = (f_i, g_i)ai​=(fi​,gi​)。

显然 aaa 序列中的元素互不相同，假设有一对 (i,j)(i, j)(i,j) 满足 1≤i<j≤n1 \le i < j \le n1≤i<j≤n，那么 pi>pjp_i > p_jpi​>pj​ 能推出 gi>gjg_i > g_jgi​>gj​，否则能推出 fi<fjf_i < f_jfi​<fj​。并且根据题目限制，显然有二元组 (x,y)(x, y)(x,y) 合法当且仅当 x≤A,y≤B,x+y−1≤Cx \le A, y \le B, x + y - 1 \le Cx≤A,y≤B,x+y−1≤C，aaa 序列中所有元素都应当合法。

那么我们得到了一个 nnn 的上界 AB−(A+B−C2)AB - \binom{A + B - C}2AB−(2A+B−C​)，我们不妨猜测 nnn 一定能取到它。后面的过程中将会证明这一点。

考虑任意一个合法二元组 (x,y)(x, y)(x,y)，如果有 x>1x > 1x>1，那么 (x−1,y)(x - 1, y)(x−1,y) 在 aaa 中的位置 iii 一定在 (x,y)(x, y)(x,y) 在 aaa 中的位置 jjj 之前，且满足 pi<pjp_i < p_jpi​<pj​。

  • 若 i>ji > ji>j，则根据 fi<fjf_i < f_jfi​<fj​ 推出 fjf_jfj​ 不能转移到 fif_ifi​，也就是 pj>pip_j > p_ipj​>pi​，这样 gig_igi​ 就能转移到 gjg_jgj​，与 gi=gjg_i = g_jgi​=gj​ 这一条件矛盾。
  • 那么有 i<ji < ji<j，则根据 gi=gjg_i = g_jgi​=gj​ 推出 gjg_jgj​ 不能转移到 gig_igi​，所以 pi<pjp_i < p_jpi​<pj​。

同理，如果有 y>1y > 1y>1，那么 (x,y−1)(x, y - 1)(x,y−1) 在 aaa 中的位置 iii 一定在 (x,y)(x, y)(x,y) 在 aaa 的位置 jjj 之后，且满足 pi<pjp_i < p_jpi​<pj​。

可以发现上述条件是长为 nnn 的排列 ppp 合法的充要条件，那么这个排列 ppp 与两个相同形状的 nnn 格杨表组成的杨表对 (λ,μ)(\lambda, \mu)(λ,μ) 构成双射：

  • 两个杨表都包含了所有坐标形如 (i,j)(i, j)(i,j) 的格子，满足 (i,j)(i, j)(i,j) 为一个合法二元组，且这两个杨表中 1∼n1 \sim n1∼n 恰好出现一次。
  • 对于第一个杨表，有 λi,j<λi,j+1,λi,j<λi+1,j\lambda_{i, j} < \lambda_{i, j + 1}, \lambda_{i, j} < \lambda_{i + 1, j}λi,j​<λi,j+1​,λi,j​<λi+1,j​。
  • 对于第二个杨表，有 μi,j<μi,j−1,μi,j<μi+1,j\mu_{i, j} < \mu_{i, j - 1}, \mu_{i, j} < \mu_{i + 1, j}μi,j​<μi,j−1​,μi,j​<μi+1,j​。
