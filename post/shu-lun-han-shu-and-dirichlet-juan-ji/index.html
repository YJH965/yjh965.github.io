数论函数

在数论上，数论函数指定义域为正整数的函数。

积性与完全积性

如果一个数论函数 fff 满足 ∀n,m≥1,gcd⁡(n,m)=1 f(nm)=f(n)f(m)\forall n, m \ge 1, \gcd(n, m) = 1 \ f(nm) = f(n)f(m)∀n,m≥1,gcd(n,m)=1 f(nm)=f(n)f(m)，则称这个函数是积性的。

如果一个数论函数 fff 满足 ∀n,m≥1 f(nm)=f(n)f(m)\forall n, m \ge 1 \ f(nm) = f(n)f(m)∀n,m≥1 f(nm)=f(n)f(m)，则称这个函数是完全积性的。

根据定义，所有完全积性函数的集合是所有积性函数的集合的子集。

一些重要的数论函数

  • φ(n)\varphi(n)φ(n) 表示多少个数小于等于 nnn 且与 nnn 互质。
  • τ(n)\tau(n)τ(n) 表示 nnn 的约数个数。
  • σ(n)\sigma(n)σ(n) 表示 nnn 的约数之和。
  • dk(n)d_k(n)dk​(n) 表示约数 kkk 次方和，即 ∑d∣ndk\sum_{d | n} d^k∑d∣n​dk。且 τ=d0,σ=d1\tau = d_0, \sigma = d_1τ=d0​,σ=d1​。
  • μ(n)\mu(n)μ(n) 对于有平方因子的数为 000，否则有奇数 / 偶数个质因数时为 −1/1-1 / 1−1/1。
  • Idk(n)\mathrm{Id}_k(n)Idk​(n) 为 nkn^knk。
  • ϵ(n)\epsilon(n)ϵ(n) 为 [n=1][n = 1][n=1]。

可以证明，以上数论函数都是积性函数。

线性筛求积性函数

根据积性函数的定义，只要我们可以求出所有 f(pk)f(p^k)f(pk) 的值（ppp 为质数，kkk 为正整数），我们就可以用线性筛筛出积性函数 fff。

举几个例子：假设 nnn 的质因数分解为 n=∏i=1mpiain = \prod_{i = 1}^m {p_i}^{a_i}n=∏i=1m​pi​ai​，那么：

  • 根据积性函数的性质，φ(n)=∏i=1mφ(piai)=∏i=1m(piai−piai−1)\varphi(n) = \prod_{i = 1}^m \varphi({p_i}^{a_i}) = \prod_{i = 1}^m ({p_i}^{a_i} - {p_i}^{a_i - 1})φ(n)=∏i=1m​φ(pi​ai​)=∏i=1m​(pi​ai​−pi​ai​−1)。
  • 根据积性函数的性质，τ(n)=∏i=1mτ(piai)=∏i=1m(ai+1)\tau(n) = \prod_{i = 1}^m \tau({p_i}^{a_i}) = \prod_{i = 1}^m (a_i + 1)τ(n)=∏i=1m​τ(pi​ai​)=∏i=1m​(ai​+1)。
  • 根据积性函数的性质，σ(n)=∏i=1mσ(piai)=∏i=1m(1+pi+pi2+⋯+piai)\sigma(n) = \prod_{i = 1}^m \sigma({p_i}^{a_i}) = \prod_{i = 1}^m (1 + p_i + {p_i}^2 + \cdots + {p_i}^{a_i})σ(n)=∏i=1m​σ(pi​ai​)=∏i=1m​(1+pi​+pi​2+⋯+pi​ai​)。

在 φ\varphiφ 中，φ(pk)=pk−pk−1\varphi(p^k) = p^k - p^{k - 1}φ(pk)=pk−pk−1；在 τ\tauτ 中，τ(pk)=k+1\tau(p^k) = k + 1τ(pk)=k+1；在 σ\sigmaσ 中，σ(pk)=1+p+p2+⋯+pk\sigma(p^k) = 1 + p + p^2 + \cdots + p^kσ(pk)=1+p+p2+⋯+pk。

这三个值都可以计算，所以它们可以用线性筛筛出。

那效率怎么样呢？我们知道，线性筛递推一次的时间复杂度为 O(n)O(n)O(n)，而 1∼n1 \sim n1∼n 中大概有 nlog⁡n\dfrac{n}{\log n}lognn​ 个形如 pkp^kpk 的数，设求 f(pk)f(p^k)f(pk) 的时间复杂度为 ttt，那么总时间复杂度为 O(n+ntlog⁡n)O(n + \dfrac{nt}{\log n})O(n+lognnt​)。

所以，如果 ttt 至多为为 O(log⁡n)O(\log n)O(logn) 级别，时间复杂度为线性。

线性筛出 φ\varphiφ 并做前缀和即可线性求出答案。

Dirichlet 卷积

对于两个数论函数 f,gf, gf,g，它们的 Dirichlet 卷积 h=f∗gh = f * gh=f∗g 被定义为 h(n)=∑d∣nf(d)g(n/d)h(n) = \sum_{d | n} f(d)g(n / d)h(n)=∑d∣n​f(d)g(n/d)。

这个卷积满足交换律和结合律，则 f∗g=g∗f,(f∗g)∗h=f∗(g∗h)f * g = g * f, (f * g) * h = f * (g * h)f∗g=g∗f,(f∗g)∗h=f∗(g∗h)。

关于上面的数论函数，有一些性质：

  1. ϵ∗f=f\epsilon * f = fϵ∗f=f，其中 fff 为任意数论函数。（即，ϵ\epsilonϵ 是 Dirichlet 卷积的单位元）

    证明

    把卷积的式子写出来是这样的：∑d∣nϵ(d)f(n/d)\sum_{d | n} \epsilon(d)f(n / d)∑d∣n​ϵ(d)f(n/d)。只有 d=1d = 1d=1 时 ϵ(d)=1\epsilon(d) = 1ϵ(d)=1，所以成立。

  2. dk=Idk∗Id0d_k = \mathrm{Id}_k * \mathrm{Id}_0dk​=Idk​∗Id0​。（我们还可以推出 τ=Id0∗Id0,σ=Id1∗Id0\tau = \mathrm{Id}_0 * \mathrm{Id}_0, \sigma = \mathrm{Id}_1 * \mathrm{Id}_0τ=Id0​∗Id0​,σ=Id1​∗Id0​）

    证明

    把卷积的式子写出来是这样的：∑i∣nik=dk(n)\sum_{i | n} i^k = d_k(n)∑i∣n​ik=dk​(n)，与 dkd_kdk​ 的定义一致。

  3. φ∗Id0=Id1\varphi * \mathrm{Id}_0 = \mathrm{Id}_1φ∗Id0​=Id1​。

    证明

    想证明这个，其实就是证明 ∑d∣nφ(d)=n\sum_{d | n} \varphi(d) = n∑d∣n​φ(d)=n。

    我们考虑下列 nnn 个分数：1n,2n,…,nn\dfrac 1n, \dfrac 2n, \dots, \dfrac nnn1​,n2​,…,nn​。

    把它们都约分至最简形式。显然约分之后的分母只会是 nnn 的因数。

    对于每个因数 xxx，这些分数里分母为 xxx 的分数显然有 φ(x)\varphi(x)φ(x) 个。而且所有分数一共有 nnn 个，所以原命题成立。

  4. μ∗Id0=ϵ\mu * \mathrm{Id}_0 = \epsilonμ∗Id0​=ϵ（非常重要！）

    证明

    如果 nnn 可以被质因数分解为 n=∏i=1mpiain = \prod_{i = 1}^m {p_i}^{a_i}n=∏i=1m​pi​ai​，那么 nnn 的一个约数 ddd 可以被质因数分解为 d=∏i=1mpibid = \prod_{i = 1}^m {p_i}^{b_i}d=∏i=1m​pi​bi​，且 ∀i∈[1,m] 0≤bi≤ai\forall i \in [1, m] \ 0 \le b_i \le a_i∀i∈[1,m] 0≤bi​≤ai​。

    如果有任意一个 bi≥2b_i \ge 2bi​≥2，则 μ(d)=0\mu(d) = 0μ(d)=0，这是没有意义的，所以我们只考虑所有 bi≤1b_i \le 1bi​≤1 的情况。

    如果 m≥1m \ge 1m≥1（即 n>1n > 1n>1），bib_ibi​ 之和为奇数与和为偶数的情况是一样的，都有 2m−12^{m - 1}2m−1 种，所以会抵消，即 ∑d∣nμ(d)=0\sum_{d | n} \mu(d) = 0∑d∣n​μ(d)=0。

    否则，∑d∣1μ(d)\sum_{d | 1} \mu(d)∑d∣1​μ(d) 即为 μ(1)=1\mu(1) = 1μ(1)=1。

    这与 ϵ\epsilonϵ 的定义一致，原命题成立。

根据前面的 μ∗Id0=ϵ\mu * \mathrm{Id}_0 = \epsilonμ∗Id0​=ϵ，可以得出 [n=1]=∑d∣nμ(d)[n = 1] = \sum_{d | n} \mu(d)[n=1]=∑d∣n​μ(d)，这可以帮助推出题目式子。
