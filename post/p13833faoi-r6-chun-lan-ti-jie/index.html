P13833「FAOI-R6」纯蓝 题解

2025-08-25 题解

挺好一个题。T4 漏了一万次情况导致没时间做这个了，赛时只写了带 log⁡\loglog 做法。

考虑把最小异或和之和拆开，设 F(k)F(k)F(k) 为选出一个子集使得最小异或和 >k> k>k 的方案数，那么答案即为 ∑i=0F(k)\sum_{i = 0} F(k)∑i=0​F(k)。可以发现 kkk 的上界是 O(mn)\mathcal{O}(\frac mn)O(nm​) 级别的，其中 mmm 为值域。这是因为，如果我们将 kkk 收缩到最大的形如 2p−12^p - 12p−1 的数，那么条件等价于 ⌊ai2p⌋\left \lfloor \frac{a_i}{2^p} \right \rfloor⌊2pai​​⌋ 互不相同，于是有 2p≤mn2^p \le \frac mn2p≤nm​，可以推出 k<2p+1≤2mnk < 2^{p + 1} \le \frac{2m}nk<2p+1≤n2m​。

考虑如何计算 F(k)F(k)F(k)。根据最小异或和只可能在数值上相邻两个数之间产生的经典结论，我们考虑按值域从大到小来 DP。设 fi,jf_{i, j}fi,j​ 表示目前已经确定了前 iii 大的数，第 iii 大的数为 jjj 时的方案数，并预处理出 cntxcnt_xcntx​ 为 ∑i=1n[x≤li]\sum_{i = 1}^n [x \le l_i]∑i=1n​[x≤li​]，有转移方程：

fi,j←(cntx−i+1)fi−1,x (x⊕j>k,x>j)f_{i, j} \gets (cnt_x - i + 1) f_{i - 1, x} \ (x \oplus j> k, x > j) fi,j​←(cntx​−i+1)fi−1,x​ (x⊕j>k,x>j)

对于单个 kkk 做一遍是 O(nm2)\mathcal{O}(nm^2)O(nm2)，总时间复杂度为 O(m3)\mathcal{O}(m^3)O(m3)。考虑优化。

可以发现满足 x⊕j>kx \oplus j > kx⊕j>k 的 xxx 在值域上会形成 O(log⁡m)\mathcal{O}(\log m)O(logm) 个区间，具体来说我们枚举 x⊕jx \oplus jx⊕j 是在哪一位严格大于 kkk 的即可。那么我们可以将单个 kkk 的时间复杂度优化到 O(nmlog⁡m)\mathcal{O}(nm \log m)O(nmlogm)，总时间复杂度为 O(m2log⁡m)\mathcal{O}(m^2 \log m)O(m2logm)。

这是求解单个 F(k)F(k)F(k) 的参考代码：

int solve(int x){int ans = 0;
	for (int i = 0;i < N;i ++) f[i] = a[i];
	for (int i = 1;i < n;i ++){
		for (int j = 0;j < N;j ++) s[j] = f[j], f[j] = 0;
		for (int j = 1;j < N;j ++) add(s[j], s[j - 1]);
		for (int j = 0;j < N;j ++) if (a[j] > i){int tp = 0;
			for (int k = D - 1;k >= 0;k --){
				if (j >> k & 1 ^ 1) tp ^= 1 << k;
				if (x >> k & 1 ^ 1)
				add(f[j], S(max(j + 1, tp), tp + (1 << k) - 1)), tp ^= 1 << k;
			}mul(f[j], a[j] - i);
		}

}

实际上这个东西跑的很快，最慢点 2.07s，卡一下肯定能过。

但是我们可以做到更优。注意到我们在令 j←j+1j \gets j + 1j←j+1 的时候实际上是对于 jjj 的二进制表示的一段后缀进行了反转，而这 O(log⁡m)\mathcal{O}(\log m)O(logm) 个区间中也只有被反转的位对应的区间会受影响。显然被反转的后缀的总长度是 O(m)\mathcal{O}(m)O(m) 级别的，我们在每次 j←j+1j \gets j + 1j←j+1 的时候重新处理受影响的区间即可，那么总时间复杂度为 O(m2)\mathcal{O}(m^2)O(m2)。

这是求解单个 F(k)F(k)F(k) 的参考代码：

int solve(int x){int ans = 0;
	for (int i = 0;i < N;i ++) f[i] = a[i];
	for (int i = 1;i < n;i ++){
		for (int j = 0;j < N;j ++) s[j] = f[j], f[j] = 0;
		for (int j = 1;j < N;j ++) add(s[j], s[j - 1]);
		int tp = 0, now = 0;
		for (int k = D - 1;k >= 0;k --){tp ^= 1 << k;
			if (x >> k & 1 ^ 1)
			add(now, S(max(1, tp), tp + (1 << k) - 1)), tp ^= 1 << k;
		}
		for (int j = 0, p;a[j] > i;){
			mul(f[j] = now, a[j] - i), p = __builtin_ctz(j + 1);
			for (int k = 0;k <= p;k ++){
				if (x >> k & 1 ^ 1)
				tp ^= 1 << k, add(now, mod - S(max(j + 1, tp), tp + (1 << k) - 1));
				if (j >> k & 1 ^ 1) tp ^= 1 << k;
			}j ++;
			for (int k = p;k >= 0;k --){
				if (j >> k & 1 ^ 1) tp ^= 1 << k;
				if (x >> k & 1 ^ 1)
				add(now, S(max(j + 1, tp), tp + (1 << k) - 1)), tp ^= 1 << k;
			}
		}
	}
	for (int i = 0;i < N;i ++) add(ans, f[i]);
	return ans;
}
