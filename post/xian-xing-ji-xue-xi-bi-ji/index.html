线性基学习笔记

2023-04-16 学习笔记

线性基

定义

张成

一个集合 sss 的所有非空子集的异或和所组成的集合为 sss 的张成，记作 span⁡(s)\operatorname{span}(s)span(s)。

线性相关 & 线性无关

如果一个集合 sss 存在一个元素 xxx 使得 sss 去掉 xxx 之后形成的集合 s′s's′ 满足 x∈span⁡(s′)x \in \operatorname{span}(s')x∈span(s′)，则称 sss 线性相关。

其实就是 sss 存在一个元素可以由其它几个元素异或得到，而且从这可以知道，去掉一个线性相关的集合 sss 里像 xxx 一样的元素后，sss 的张成不变。

反之，则称集合 sss 线性无关。

线性基

我们称 ttt 是 sss 的线性基，当且仅当：

  • s⊆span⁡(t)s \subseteq \operatorname{span}(t)s⊆span(t)；
  • ttt 线性无关。

性质

sss 的线性基 ttt 有两个基本性质：

  • ttt 不存在一个真子集 uuu 使得 uuu 是 sss 的线性基。

    因为 ttt 线性无关，所以无法去掉 ttt 的任意元素使得 ttt 的张成不变。

  • sss 的每个元素都可以被唯一表示为 ttt 的若干元素的异或和。

    首先因为 s⊆span⁡(t)s \subseteq \operatorname{span}(t)s⊆span(t)，所以 sss 的每个元素都可以被表示为 ttt 的若干元素的异或和。

    接下来使用反证法证明这个表示是唯一的。

    假设 sss 中一个元素 xxx 等于 ttt 的子集 uuu 的异或和，也等于子集 vvv 的异或和，且 uuu 和 vvv 没有交集。（uuu 和 vvv 的公共部分是没有意义的）

    然后 uuu 中的一个元素 yyy 可以被自己异或得到，也可以被 uuu 里的剩余元素和 vvv 的所有元素异或而成，这不满足 ttt 的线性无关性。

构造

我们使用一个数组 bbb 存储线性基，当 bi≠0b_i \ne 0bi​​=0 时我们称第 iii 位存在。

这样构造出来的线性基有一个有用的限制：每个元素最高位互不相同。

我们考虑将数一个一个插入线性基。

根据线性基的性质，nnn 个元素中，只要有一个元素第 iii 位为 111，第 iii 位就存在。所以如果数据随机，线性基的大小就比较大，这个性质有时候会应用。

插入过程

设插入的数为 xxx，考虑从高位考虑到低位，假设现在考虑到第 iii 位。

  • 如果 xxx 的第 iii 位为 000，直接进入下一位的考虑即可。
  • 否则：
    • 如果第 iii 位存在，我们需要将 xxx 的第 iii 位消除，（因为只有 bib_ibi​ 的第 iii 位为 111）把 xxx 异或上 bib_ibi​ 即可。
    • 如果第 iii 位不存在，我们将 xxx 插入 bbb。

如果 xxx 自始至终都没有被插入，那就说明 xxx 可以被线性基里的一些数异或出来，如果插入就不满足线性无关性。

证明

因为我们对 xxx 以及 bbb 的操作都是互相异或，这些操作显然是可逆的，所以用 bbb 的一些元素的异或和可以表示出原本的 xxx。

代码

void insert(ll x){
	for (int i = 49;i >= 0;i --){
		if (x >> i & 1 ^ 1) continue;
		if (b[i]) x ^= b[i]; // 用 b[i] 消去 x 的第 i 位
		else { // 可以插入
			b[i] = x; // 插入
			break; // 一定不要忘记
		}
	}
    // 如果没被插入（即 x 被异或成 0）就说明不需要 x
}
