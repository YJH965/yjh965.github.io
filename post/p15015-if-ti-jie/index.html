题意简述

定义一个字符串 aaa 中子序列 bbb 的数量为满足 ∀i,api=bi\forall i, a_{p_i} = b_i∀i,api​​=bi​ 的长度为 ∣b∣|b|∣b∣，值域为 [1,∣a∣][1, |a|][1,∣a∣] 的严格单调递增正整数序列 ppp 的个数。

定义一个只包含 L、I、F、E 四种字符的字符串是好的，当且仅当这个字符串中只存在恰好一个子序列 IF 与一个子序列 LIE。

请你求出 AAA 个字符 L，BBB 个字符 I，CCC 个字符 F，DDD 个字符 E 组成的共 (A+B+C+D)!A!B!C!D!\frac{(A + B + C + D)!}{A!B!C!D!}A!B!C!D!(A+B+C+D)!​ 种不同的字符串中好的字符串的数量对 109+710^9 + 7109+7 取模的值。

此题多测，设 TTT 为组数，对所有数据，满足 1≤T≤2×1051 \le T \le 2 \times 10^51≤T≤2×105，1≤A,B,C,D≤1071 \le A, B, C, D \le 10^71≤A,B,C,D≤107，1≤∑B≤2×1071 \le \sum B \le 2 \times 10^71≤∑B≤2×107。

我会 DP！

考虑如何判断字符串 aaa 中子序列 bbb 是否出现了一次。我们可以这样做：

  • 维护一个指针 jjj，初始 j=0j = 0j=0。
  • 从 111 到 ∣a∣|a|∣a∣ 按顺序枚举 iii：
    • 若 j<∣b∣j < |b|j<∣b∣ 且 ai=bj+1a_i = b_{j + 1}ai​=bj+1​，使 j←j+1j \gets j + 1j←j+1。
    • 否则若 ai=bja_i = b_jai​=bj​，则必定不满足条件。
  • 最后若 j=∣b∣j = |b|j=∣b∣ 则满足，否则不满足。

正确性显然。那么我们直接设 fa,b,c,d,x,yf_{a, b, c, d, x, y}fa,b,c,d,x,y​ 表示目前 L、I、F、E 的数量分别为 a,b,c,da, b, c, da,b,c,d，对于 IF 的指针为 xxx，对于 LIE 的指针为 yyy 时的答案，最后输出 fA,B,C,D,2,3f_{A, B, C, D, 2, 3}fA,B,C,D,2,3​ 即可。

算法 4

我会观察形态！

  • 根据上面的判断方式，只考虑 I、F 时字符串应当形如“一段 F，IF，一段 I”的形式，其中首尾这两段都可以为空。
  • 只考虑 L、I、E 时字符串应当形如“一段 I/E，L，一段 E、I、一段 L、E、一段 L/I”的形式，其中每一个段都可以为空。

将两者结合起来，根据只考虑 L、I、E 时字符串第一段是否存在 I 可以讨论出四种情况：

  • 字符串形如“一段 F/E，L，一段 F/E、I、一段 L、E、一段 L、F、一段 L/I”的形式，其中每一个段都可以为空。
  • 字符串形如“一段 F/E，L，一段 F/E、I、一段 L、F、一段 L、E、一段 L/I”的形式，其中每一个段都可以为空。
  • 字符串形如“一段 F/E，I，一段 E、L、一段 E、F、一段 E、I、一段 L、E、一段 L/I”的形式，其中每一个段都可以为空。
  • 字符串形如“一段 F/E，I，一段 E、F、一段 I/E、L、一段 E、I、一段 L、E、一段 L/I”的形式，其中每一个段都可以为空。

发现这四种情况的每个前缀都只有最多两种字母不是“要么几乎没填过，要么几乎全填上了”的状态，所以前面 DP 的有效状态数就是平方级别的。

前两种情况显然是等价的，算出来第一种之后对答案乘以二即可。

对于这两种情况的单个 I 的位置，显然它已经固定了，由于除了右边单个 E 和单个 F，其他的 E 和 F 都在其左边；除了左边的单个 L，其他的 L 和 I 都在其右边，于是这个 I 的位置就是 C+DC + DC+D，而它左右两边互不干涉。它的左边是 C−1C - 1C−1 个 F、一个 L、D−1D - 1D−1 个 E 任意排列的结果，方案数为 (C+D−1)!(C−1)!(D−1)!\frac{(C + D - 1)!}{(C - 1)!(D - 1)!}(C−1)!(D−1)!(C+D−1)!​；它的右边是 A−1A - 1A−1 个 L、一个 F、一个 E、B−1B - 1B−1 个 I 任意排列的结果，并且要求 E 在 F 之前，F 在所有 I 之前，那我们可以将其看作 A−1A - 1A−1 个 L 与 B+1B + 1B+1 个 I 任意排列，并将前两个 I 分别替换成 E 和 F，方案数为 (A+BA−1)\binom{A + B}{A - 1}(A−1A+B​)，两边乘起来再乘上 222 即可。

对于 B=1B = 1B=1 的特殊性质，后两种情况是不可能出现的，所以只需要分析到这里就可以获得对应分数。

对于第三种情况的分析也是类似的，方案数为 (C+D+1D−1)(A+B−2A−1)\binom{C + D + 1}{D - 1} \binom{A + B - 2}{A - 1}(D−1C+D+1​)(A−1A+B−2​)。

最后一种情况比较特殊，它没有前三种情况这么好的性质，所以我们直接枚举中间那个 I/E 段中 I 的个数 iii（0≤i≤B−20 \le i \le B - 20≤i≤B−2），即可确定后面那个单个 I 的位置为 C+D+i+2C + D + i + 2C+D+i+2，它左右两边互不干涉。它的左边是 D−1D - 1D−1 个 E 与剩下的东西归并起来的结果，方案数为 (C+D+i+1D−1)\binom{C + D + i + 1}{D - 1}(D−1C+D+i+1​)；它的右边是 A−1A - 1A−1 个 L 与剩下的东西归并起来的结果，方案数为 (A+B−i−2A−1)\binom{A + B - i - 2}{A - 1}(A−1A+B−i−2​)。

那么我们就算完了所有情况的答案，瓶颈在最后一种情况的枚举。

时间复杂度：O(∑B)\mathcal O(\sum B)O(∑B)。

期望得分：100100100。
