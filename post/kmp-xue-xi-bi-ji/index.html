KMP 算法，又称模式匹配算法，可以在线性时间内判断字符串 AAA 是否是字符串 BBB 的子串，还可以查找 AAA 在 BBB 中出现的位置。

KMP 算法分为两步：

  1. 对 AAA 进行自我匹配，求出数组 nextnextnext，其中 nextinext_inexti​ 是满足 nexti<inext_i < inexti​<i 且满足 A[1,nexti]=A[i−nexti+1,i]A[1, next_i] = A[i - next_i + 1, i]A[1,nexti​]=A[i−nexti​+1,i] 的最大值（特别的，如果不存在 nextinext_inexti​，则 nexti=0next_i = 0nexti​=0）。

代码

模板题代码：

#include <cstring>
#include <cstdio>

using namespace std;

int a, b, nxt[1000001];
char s1[1000001], s2[1000001];
int main(){
	scanf("%s%s", s1 + 1, s2 + 1);
	a = strlen(s1 + 1), b = strlen(s2 + 1);
	nxt[1] = 0;
	for (int i = 2, j = 0;i <= b;i ++){
		while (j && s2[j + 1] != s2[i]) j = nxt[j]; // 不断失配
		if (s2[j + 1] == s2[i]) j ++; // 匹配成功
		nxt[i] = j;
	} // 自我匹配
	for (int i = 1, j = 0;i <= a;i ++){
		while (j && s2[j + 1] != s1[i]) j = nxt[j];
		if (s2[j + 1] == s1[i]) j ++;
		if (j == b) printf("%d\n", i - b + 1), j = nxt[j]; // A 在 B 中出现
	}
	for (int i = 1;i <= b;i ++) printf("%d ", nxt[i]);
}
