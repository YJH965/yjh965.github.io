决策单调性优化 DP

本文参考了这篇博客进行写作。下文的“优”均代指更小。

四边形不等式

定义：设 cost(i,j)cost(i, j)cost(i,j) 为代价函数，那么其满足四边形不等式，当且仅当 ∀a≤b≤c≤d,cost(a,c)+cost(b,d)≤cost(a,d)+cost(b,c)\forall a \le b \le c \le d, cost(a, c) + cost(b, d) \le cost(a, d) + cost(b, c)∀a≤b≤c≤d,cost(a,c)+cost(b,d)≤cost(a,d)+cost(b,c)，即"交叉优于包含"。

优化部分 1D/1D DP

考虑 DP：fi=min⁡{fj+cost(j,i)}f_i = \min\{f_j + cost(j, i)\}fi​=min{fj​+cost(j,i)}。以下设 pip_ipi​ 为 fif_ifi​ 的最优转移点（如果有多个则强制钦定是最左 / 最右的）。

  • 有结论：若 costcostcost 满足四边形不等式，则 fif_ifi​ 满足决策单调性（即 p1,p2,…,pnp_1, p_2, \dots, p_np1​,p2​,…,pn​ 递增）。

    证明考虑反证，假设存在 a,b,c,da, b, c, da,b,c,d 满足 a<b<c<da < b < c < da<b<c<d，pc=b,pd=ap_c = b, p_d = apc​=b,pd​=a。那么据此显然有：

那么根据决策单调性，我们接下来会有多种解决方式。

分治优化

直接设 solve(l,r,L,R)\mathrm{solve}(l, r, L, R)solve(l,r,L,R) 表示 L≤pl,pr≤RL \le p_l, p_r \le RL≤pl​,pr​≤R 的转移。直接设 mid=⌊l+r2⌋mid = \lfloor\frac{l+r}2\rfloormid=⌊2l+r​⌋，暴力找到它的转移点 pmidp_{mid}pmid​，递归到 solve(l,mid−1,L,pmid),solve(mid+1,r,pmid,R)\mathrm{solve}(l, mid - 1, L, p_{mid}), \mathrm{solve}(mid + 1, r, p_{mid}, R)solve(l,mid−1,L,pmid​),solve(mid+1,r,pmid​,R) 即可。每层中 r−l+R−Lr - l + R - Lr−l+R−L 总和是 O(n)O(n)O(n) 的，那么总转移次数是 O(nlog⁡n)O(n \log n)O(nlogn)。但是由于要先找到 midmidmid 的转移点，所以只能做到离线而不是半在线。

顺带一提，若 costcostcost 不能快速维护，那么用类莫队的方法移动指针时，总移动次数也是 O(nlog⁡n)O(n \log n)O(nlogn) 的。应该也可以支持回滚。

二分栈优化

考虑直接从左到右扫，并维护合法的转移点。由于有决策单调性，那么我们新加入一个转移点时一定是只有一个后缀可能被更新。那么我们直接考虑颜色段均摊。维护一个三元组 (l,r,o)(l, r, o)(l,r,o) 的栈，其中一个三元组表示 ∀i∈[l,r],pi=o\forall i \in [l, r], p_i = o∀i∈[l,r],pi​=o。那么我们初始向栈中插入 (1,n,1)(1, n, 1)(1,n,1)，然后依次插入转移点 2∼n2 \sim n2∼n。当插入一个转移点时，栈顶的一个三元组要么被完全干掉，要么被干掉一部分，具体是哪种情况可以直接判断。如果是后者，还需要二分出具体的分界点（有时代价函数特殊时可以不需要二分）。总转移次数也是 O(nlog⁡n)O(n \log n)O(nlogn)，是半在线的。

wqs 二分优化

有时我们还对转移的次数有限制，即 fi,k=min⁡{fj,k−1+cost(j,i)}f_{i, k} = \min\{f_{j, k - 1} + cost(j, i)\}fi,k​=min{fj,k−1​+cost(j,i)}。

  • 有结论：若 costcostcost 满足四边形不等式，则 fn,kf_{n, k}fn,k​ 关于 kkk 是凸的。

    考虑证明 fn,k−1+fn,k+1≥2fn,kf_{n, k - 1} + f_{n, k + 1} \ge 2f_{n, k}fn,k−1​+fn,k+1​≥2fn,k​（这里是下凸，上凸也是同理）。考虑 fn,k−1f_{n, k - 1}fn,k−1​ 的过程中有 kkk 个转移点，fn,k+1f_{n, k + 1}fn,k+1​ 有 k+2k + 2k+2 个。那么后者一定有两个相邻的转移点在前者的同一段中。假设前者的对应那段是 [l,r][l, r][l,r]，后者的两个转移点对应的一段是 [x,y][x, y][x,y]，那么一定有 l<x≤y<rl < x \le y < rl<x≤y<r。

    考虑构造两个划分为 kkk 段的方案，具体地，将 xxx 对应的转移点从 fn,k+1f_{n, k + 1}fn,k+1​ 挪到 fn,k−1f_{n, k - 1}fn,k−1​ 中。设这两段的答案和为 sumsumsum，那么 sum≥2fn,ksum \ge 2f_{n, k}sum≥2fn,k​，所以我们只需要证明 fn,k−1+fn,k+1≥sumf_{n, k - 1} + f_{n, k + 1} \ge sumfn,k−1​+fn,k+1​≥sum 即可。将两边共同部分消掉，那么会剩下一个 cost(l−1,r)+cost(x−1,y)≥cost(l−1,y)+cost(x−1,r)cost(l -1, r) + cost(x - 1, y) \ge cost(l - 1, y) + cost(x - 1, r)cost(l−1,r)+cost(x−1,y)≥cost(l−1,y)+cost(x−1,r)，我们发现这就是四边形不等式。

优化区间 DP

考虑 DP：fl,r=min⁡{fl,k+fk,r+cost(l,r)}f_{l, r} = \min\{f_{l, k} + f_{k, r} + cost(l, r)\}fl,r​=min{fl,k​+fk,r​+cost(l,r)}，以下设 pl,rp_{l, r}pl,r​ 为 fl,rf_{l, r}fl,r​ 的最优转移点（如果有多个则强制钦定是最左 / 最右的）。

  • 有结论：如果 costcostcost 满足 ∀a≤b≤c≤d,cost(b,c)≤cost(a,d)\forall a \le b \le c \le d, cost(b, c) \le cost(a, d)∀a≤b≤c≤d,cost(b,c)≤cost(a,d)（即“小区间优于大区间”）且满足四边形不等式，那么 fl,rf_{l, r}fl,r​ 也满足四边形不等式。

    证明比较复杂，大概是讨论 pa,dp_{a, d}pa,d​ 的位置，这里就省略了。

反四边形不等式

类比四边形不等式，将其中的不等号反过来，即"包含优于交叉"。

我们考虑四边形不等式实际上是证明了所有区间 [pi,i][p_i, i][pi​,i] 不出现包含情况，那么我们直接套用上述证明过程即可得到所有区间 [pi,i][p_i, i][pi​,i] 不出现交叉（相交但不包含）情况。

二分栈优化

定义 pos(i,j)pos(i, j)pos(i,j)（i<ji < ji<j）为一个 kkk，其中对于 [j+1,k][j + 1, k][j+1,k] 都满足 jjj 作为转移点更优，对于 [k+1,n][k + 1, n][k+1,n] 都满足 iii 作为转移点更优。由于 costcostcost 满足反四边形不等式，所以一定存在这样的 kkk，并且我们可以二分出它。

考虑维护一个转移点构成的栈，同样是依次加入转移点。那么当我们加入 iii 时，令栈顶为 tptptp，次栈顶为 spspsp，若 pos(tp,i)≤pos(sp,tp)pos(tp, i) \le pos(sp, tp)pos(tp,i)≤pos(sp,tp)，那么 tptptp 就被 spspsp 与 iii 偏序了，可以扔掉，重复迭代这个弹栈过程即可，最后判一下 iii 是否会被栈顶全面偏序，如果不会就 push 进去。

总转移次数是 O(nlog⁡n)O(n \log n)O(nlogn) 的。

线段树分治优化

该优化也对四边形不等式适用。直接对于每个 iii，将其能转移到的区间 [Li,Ri][L_i, R_i][Li​,Ri​] 对应的 log⁡\loglog 个线段树上的区间都插入一个 iii。那么对于每个线段树上的区间，能转移到该区间的节点都在该区间外，所以我们就拥有了“反决策单调性”，即 ppp 在这个区间对应的转移过程中单调递减。那么我们在线段树分治的内层直接对每个线段树表示的区间做前文提到的分治算法即可，整个过程是半在线的。

总转移次数是 O(nlog⁡2n)O(n \log^2 n)O(nlog2n) 的，该算法实质上是将离线算法套了一层分治变为半在线，并且可以处理转移区间任意的情况。
