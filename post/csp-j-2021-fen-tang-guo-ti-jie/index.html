[CSP-J 2021] 分糖果 题解

2023-04-16 杂物 ， 题解

思路

简要题意

给出 LLL，RRR，nnn，你需要在区间 [L,R][L, R][L,R] 中取一个数 kkk，使 k mod nk \bmod nkmodn 最大。

显然是道大水题。

首先，任何数 (modn)\pmod n(modn) 最大只可能为 n−1n - 1n−1。如果在 [L,R][L, R][L,R] 中能取到 n−1n - 1n−1，我们就取；否则，我们取 R mod nR \bmod nRmodn。

有 n−1n - 1n−1 就取是没什么问题，但是为什么没有 n−1n - 1n−1 时取 R mod nR \bmod nRmodn 呢？

因为，没有 n−1n - 1n−1 可取时，可以发现，L mod n<(L+1) mod n<(L+2) mod n<...<(R−1) mod n<R mod nL \bmod n < (L + 1) \bmod n < (L + 2) \bmod n < ... < (R - 1) \bmod n < R \bmod nLmodn<(L+1)modn<(L+2)modn<...<(R−1)modn<Rmodn。

也就是说，(modn)\pmod n(modn) 的结果是单调递增的。

这时，最大值肯定为 R mod nR \bmod nRmodn。

代码

#include <algorithm>
#include <iostream>
#include <cstring>
#include <cstdio>

using namespace std;

int n, l, r, x, ans;
int main(){
	scanf("%d%d%d", &n, &l, &r);
	x = r / n;
	if (n * x - 1 >= l) ans = n - 1;
	else ans = r % n;
	printf("%d", ans);
}
