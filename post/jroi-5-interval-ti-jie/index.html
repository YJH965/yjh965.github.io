『JROI-5』Interval 题解

2023-04-16 题解

思路

因为 aaa 是一个排列，所以我们可以记录 iii 出现的位置 pip_ipi​。

根据 pip_ipi​，我们可以递推出 lil_ili​ 和 rir_iri​ 满足 Mex⁡li,ri=i+1\operatorname{Mex}_{l_i, r_i} = i + 1Mexli​,ri​​=i+1 且 ri−li+1r_i - l_i + 1ri​−li​+1 最小。根据 Mex⁡\operatorname{Mex}Mex 的定义，li=min⁡(li−1,pi)l_i = \min(l_i - 1, p_i)li​=min(li​−1,pi​)，ri=max⁡(ri−1,pi)r_i = \max(r_i - 1, p_i)ri​=max(ri​−1,pi​)。

我们可以再处理出 sis_isi​ 代表区间长度为 iii 的最大 Mex⁡\operatorname{Mex}Mex 值。

Mex⁡\operatorname{Mex}Mex 值为 xxx 且长度 lll 最小，相当于 xxx 就是满足长度为 lll 的最大 Mex⁡\operatorname{Mex}Mex 值，所以对于每一个 iii，sri−li+1=i+1s_{r_i - l_i + 1} = i + 1sri​−li​+1​=i+1。

如果 L≤lL \le lL≤l 且 r≤Rr \le Rr≤R，Mex⁡L,R≥Mex⁡l,r\operatorname{Mex}_{L, R} \ge \operatorname{Mex}_{l, r}MexL,R​≥Mexl,r​，所以 sis_isi​ 还要和 max⁡j=1i−1sj\max_{j = 1}^{i - 1} s_jmaxj=1i−1​sj​ 取最大值。

处理出 sis_isi​ 后，枚举区间长度 iii，判断是否满足 si>fis_i > f_isi​>fi​ 即可。

代码

#include <algorithm>
#include <cstdio>

using namespace std;

int n, x, p[4000001], a[4000001], l, r, s[4000001], ans;
int main(){
	scanf("%d", &n);
	for (int i = 1;i <= n;i ++) scanf("%d", &x), p[x] = i;
	for (int i = 1;i <= n;i ++) scanf("%d", &a[i]);
	l = r = p[1], s[1] = 2;
	for (int i = 2;i <= n;i ++) l = min(l, p[i]), r = max(r, p[i]), s[r - l + 1] = i + 1;
	for (int i = 1;i <= n;i ++) s[i] = max(s[i - 1], s[i]);
	for (int i = 1;i <= n;i ++){
		if (s[i] > a[i]){
			printf("%d", i);
			return 0;
		}
	}
	printf("%d", 0);
}
