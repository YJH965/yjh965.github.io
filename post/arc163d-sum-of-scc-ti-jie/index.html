[ARC163D] Sum of SCC 题解

2023-07-03 题解 ， 科技？

首先，SCC 个数显然是一个很强的东西，看起来就显得很不可维护，至少不能高效的表示进状态里。

然后我们考虑把 SCC 个数转化下：

有结论：一个竞赛图的 SCC 个数等于将其点集划分为两个集合 A,BA, BA,B（可为空集）并满足以下限制的方案数 −1-1−1：

  • 对于每条满足 u∈A,v∈bu \in A, v \in bu∈A,v∈b 的边 (u,v)(u, v)(u,v)，都满足其方向为 u→vu \to vu→v。

证明这个结论也很简单，考虑将这个图缩点，然后它仍然是一个竞赛图，并且是一个 DAG。考虑它的拓扑序 p1,p2,p3,…,pkp_1, p_2, p_3, \dots, p_kp1​,p2​,p3​,…,pk​（因为它是竞赛图，所以拓扑序唯一），并找到一个分界点 i(0≤i≤k)i(0 \le i \le k)i(0≤i≤k)，将 p1,p2,…,pip_1, p_2, \dots, p_ip1​,p2​,…,pi​ 所对应的 SCC 划入 AAA，将 pi+1,pi+2,…,pkp_{i + 1}, p_{i + 2}, \dots, p_kpi+1​,pi+2​,…,pk​ 所对应的 SCC 划入 BBB。

可以发现一定只有这样划分才是合法的方案，因为把一个 SCC 分到两个集合，一个集合内拓扑序不连续的情况都是不合法的。而每个 iii 就对应了一种合法的方案，且 iii 的取值共有 k+1k + 1k+1 种，−1-1−1 则变为 kkk，即 SCC 个数。

现在我们的目标就变为对上述划分方案计数，而这个计数是简单的，直接 dp 即可：设 fi,j,kf_{i, j, k}fi,j,k​ 为加入了 1∼i+j1 \sim i + j1∼i+j 号点，且满足 ∣A∣=i,∣B∣=j|A| = i, |B| = j∣A∣=i,∣B∣=j，有 kkk 条满足要求的边的方案数。

考虑将 i+j+1i + j + 1i+j+1 号点加入到 AAA 或 BBB，转移如下：

fi+1,j,k+x←(ix)fi,j,k(0≤x≤i)fi,j+1,k+i+x←(jx)fi,j,k(0≤x≤j)f_{i + 1, j, k + x} \gets \binom ixf_{i, j, k} (0 \le x \le i)\\ f_{i, j + 1, k + i + x} \gets \binom jxf_{i, j, k} (0 \le x \le j)\\ fi+1,j,k+x​←(xi​)fi,j,k​(0≤x≤i)fi,j+1,k+i+x​←(xj​)fi,j,k​(0≤x≤j)

对第一个转移的解释：在 AAA 内加入最大点 uuu，uuu 向 BBB 内的连边都是逆向的，向 AAA 内的连边任意。钦定 xxx 条为正向的，则系数为 (ix)\binom ix(xi​)。

对第二个转移的解释：在 BBB 内加入最大点 uuu，uuu 向 AAA 内的连边都是正向的，向 BBB 内的连边任意。钦定 xxx 条为正向的，则系数为 (jx)\binom jx(xj​)。

时间复杂度为 O(n3m)O(n^3m)O(n3m)，其中状态数为 O(n2m)O(n^2m)O(n2m)，转移为 O(n)O(n)O(n)。
