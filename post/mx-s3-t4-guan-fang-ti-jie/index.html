出题人题解。

首先考虑一个最基本的策略：

假设我们要 check pk≥Xp_k \ge Xpk​≥X 是否合法。

那么我们只需要对于所有的 1≤l≤m,l≠k1 \le l \le m, l \ne k1≤l≤m,l​=k，分别令 pk=X,pl=1−Xp_k = X, p_l = 1 - Xpk​=X,pl​=1−X，其他的 ppp 为 000，都 check 一次即可。

证明是显然的，如果我们要在 pk=Xp_k = Xpk​=X 的情况下分配其他的 ppp 使得第 iii 个学生的加权总分超过第 jjj 个学生的加权总分 (ai,k<aj,k)(a_{i, k} < a_{j, k})(ai,k​<aj,k​)，那么将剩下的 1−X1 - X1−X 集中在 ai,l−aj,la_{i, l} - a_{j, l}ai,l​−aj,l​ 最大的第 lll 个学科上一定不劣。

于是根据上述策略，我们得到了一个单组数据 O(m2nlog⁡nlog⁡v)O(m^2 n \log n \log v)O(m2nlognlogv) 并且需要二分的做法，但是答案需要输出有理数，无法二分。

考虑不排序而是直接暴力枚举 i,j,k,l(ai,k<aj,k,ai,l>aj,l)i, j, k, l (a_{i, k} < a_{j, k}, a_{i, l} > a_{j, l})i,j,k,l(ai,k​<aj,k​,ai,l​>aj,l​)，则我们得到一个不等式：

Xai,k+(1−X)ai,l≤Xaj,k+(1−X)aj,lX a_{i, k} + (1 - X) a_{i, l} \le X a_{j, k} + (1 - X) a_{j, l} Xai,k​+(1−X)ai,l​≤Xaj,k​+(1−X)aj,l​

解得：

X≥ai,l−aj,lai,l−aj,l+aj,k−ai,kX \ge \frac{a_{i, l} - a_{j, l}}{a_{i, l} - a_{j, l} + a_{j, k} - a_{i, k}} X≥ai,l​−aj,l​+aj,k​−ai,k​ai,l​−aj,l​​

于是题意就相当于对于每个 kkk，求上式的最大值，这个东西显然可以暴力 O(n2m2)O(n^2 m^2)O(n2m2) 做。

发现需要最大化的分式是 pp+q\frac p{p + q}p+qp​ 形式的，所以我们枚举 i,j,li, j, li,j,l 确定 p=ai,l−aj,lp = a_{i, l} - a_{j, l}p=ai,l​−aj,l​ 并枚举 kkk 最小化 q=aj,k−ai,kq = a_{j, k} - a_{i, k}q=aj,k​−ai,k​。

因为 qqq 只与 i,j,ki, j, ki,j,k 相关，并且 k≠lk \ne lk​=l，所以我们可以对每一对 i,ji, ji,j 枚举 kkk 来 O(n2m)O(n^2m)O(n2m) 预处理出最小次小的 qqq，即可做到 O(1)O(1)O(1) 查询 qqq。

于是我们得到了一个单组数据时间复杂度为 O(n2m)O(n^2m)O(n2m) 的做法。

m≤100m \le 100m≤100

实际上，如果我们先枚举了 k,lk, lk,l，我们就不需要对所有的 i,ji, ji,j 去判断了。根据比较的传递性，如果存在 ooo 满足 ai,k<ao,k<aj,ka_{i, k} < a_{o, k} < a_{j, k}ai,k​<ao,k​<aj,k​，则无需考虑 i,ji, ji,j 这一对是否满足要求。于是我们对于某个 (k,l)(k, l)(k,l)，将 ai,ka_{i, k}ai,k​ 相等的 iii 划为一类，则我们只需要考虑数值相邻的两个类。

考虑固定类别之后，ai,k,aj,ka_{i, k}, a_{j, k}ai,k​,aj,k​ 都被固定了，则 pp+q\frac p{p + q}p+qp​ 中的 q=aj,k−ai,kq = a_{j, k} - a_{i, k}q=aj,k​−ai,k​ 已经被确定，则我们为了让分式最大，应该最大化 p=ai,l−aj,lp = a_{i, l} - a_{j, l}p=ai,l​−aj,l​，而 ppp 是在两个类中各选取一个数作差，ai,la_{i, l}ai,l​ 选最大，aj,la_{j, l}aj,l​ 选最小即可。

每个点只会被一个类包含，对于每个类暴力扫一遍求出最大值与最小值，时间复杂度是 O(n)O(n)O(n) 的。

于是我们又得到了一个单组数据时间复杂度为 O(nm2)O(nm^2)O(nm2) 的做法。
