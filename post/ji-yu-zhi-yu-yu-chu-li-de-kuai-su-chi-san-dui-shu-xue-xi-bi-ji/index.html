基于值域预处理的快速离散对数 学习笔记

2025-10-21 学习笔记 ， 科技？

首先单次查询离散对数可以考虑 BSGS，将 g0,gB,g2B,g3B,…,gP/B×Bg^0, g^B, g^{2B}, g^{3B}, \dots, g^{P / B \times B}g0,gB,g2B,g3B,…,gP/B×B 插入哈希表，然后枚举 i=0,1,2,…,B−1i = 0, 1, 2, \dots, B - 1i=0,1,2,…,B−1 在哈希表中查询 y×g−iy \times g^{-i}y×g−i 即可。

现在我们考虑求出 log⁡1,log⁡2,…,log⁡n\log 1, \log 2, \dots, \log nlog1,log2,…,logn 怎么做。显然这个离散对数也满足 log⁡ab=log⁡a+log⁡b\log ab = \log a + \log blogab=loga+logb，所以我们只需要求出质数位置上的离散对数即可。

那么直接套用 BSGS 的做法，预处理时间复杂度为 O(P/B)\mathcal{O}(P / B)O(P/B)，所有查询时间复杂度为 O(nBln⁡n)\mathcal{O}(\frac{nB}{\ln n})O(lnnnB​)，Pln⁡n=nB2P \ln n = n B^2Plnn=nB2，平衡得到 B=Pln⁡nnB = \sqrt{\frac{P \ln n}n}B=nPlnn​​。

考虑怎么预处理一些东西来快速回答询问。假设我们预处理了 P+1\sqrt P + 1P​+1 以内的 log⁡\loglog 值，现在要求 log⁡y\log ylogy，那么若 y≤P+1y \le \sqrt P + 1y≤P​+1 就直接返回，否则令 k,rk, rk,r 分别为 PPP 除以 yyy 的商和余数，考虑 rrr 和 y−ry - ry−r 之中一定有一个不大于 yyy 的一半，如果能够规约到对应的子问题上，就可以做到单次查询 O(log⁡P)\mathcal{O}(\log P)O(logP)。接下来有对应的两种方式：

  1. 将模数 PPP 表示为 ky+rky + rky+r，变形得到 y=P−rk=−rky = \frac{P - r}k = \frac{-r}ky=kP−r​=k−r​，那么 log⁡y=log⁡(−r)−log⁡k=log⁡(−1)+log⁡r−log⁡k\log y = \log(-r) - \log k = \log(-1) + \log r - \log klogy=log(−r)−logk=log(−1)+logr−logk。
  2. 将模数 PPP 表示为 (k+1)y−(y−r)(k + 1)y - (y - r)(k+1)y−(y−r)，变形得到 y=P+y−rk+1=y−rk+1y = \frac{P + y - r}{k + 1} = \frac{y - r}{k + 1}y=k+1P+y−r​=k+1y−r​，那么 log⁡y=log⁡(y−r)−log⁡(k+1)\log y = \log(y - r) - \log(k + 1)logy=log(y−r)−log(k+1)。

通过 y>P+1y > \sqrt P + 1y>P​+1 可以得出 k≤Pk \le \sqrt Pk≤P​，那么 log⁡k,log⁡(k+1)\log k, \log (k + 1)logk,log(k+1) 可以直接查表，选择对应的子问题递归下去即可。

回到前面预处理的部分，将 n=Pn = \sqrt Pn=P​ 代入，得到 B=P1/4ln⁡1/2PB = P^{1/4} \ln^{1/2} PB=P1/4ln1/2P。则预处理部分时间复杂度为 O(P3/4ln⁡1/2P)\mathcal{O}\left(\frac{P^{3/4}}{\ln^{1/2} P}\right)O(ln1/2PP3/4​)，可以通过。
