P15013 FUN!! 题解

2026-01-24 题解

响起乐活！

题意简述

给出长度为 nnn 的数组 aaa 以及非负整数 kkk，你需要判断是否存在一张有向图满足以下要求：

  • 图中有 nnn 个点，编号分别为 1,2,…,n1, 2, \dots, n1,2,…,n，每个点恰有一条出边（可能有自环）。

  • 对于所有的 1≤i≤n1 \le i \le n1≤i≤n，都满足从点 iii 开始沿着出边走 n+kn + kn+k 步会到达点 aia_iai​。

此题多测，对所有数据，保证 1≤ai≤n≤5×1051 \le a_i \le n \le 5 \times 10^51≤ai​≤n≤5×105，0≤k≤1090 \le k \le 10^90≤k≤109，1≤∑n≤3×1061 \le \sum n \le 3 \times 10^61≤∑n≤3×106。

算法 1

爆搜。

时间复杂度：O(nn+1)\mathcal O(n^{n + 1})O(nn+1)。

注意到 kkk 可以对 606060（111 到 nnn 的 LCM）取模，取模之后直接写 O(nn+1(n+k))\mathcal O(n^{n + 1}(n + k))O(nn+1(n+k)) 适当剪枝也能过。

期望得分：151515。

算法 2

所求的图是一片内向基环森林。那么所有点在走 n+kn + kn+k 步后都会走到其所在的内向基环树的环上。

如果 i→aii \to a_ii→ai​ 连边建一张新图，这张新图所有不在环上的点都要连到环上的点，否则方案不合法。

不在环上的 iii 的要求是一定可以满足的，只需要确定了所求的图中环的形态就可以找到环上 aia_iai​ 往后退 n+k−1n + k - 1n+k−1 步的点 bib_ibi​，把 iii 连向 bib_ibi​ 即可。接下来我们忽略这些点。

于是新图变成了一些环，我们只需要判断环长的可重集是否合法即可。所求的图显然也应该是一些环，所求的图的一个长度为 ddd 的环会在新图上会被拆分为 gcd⁡(d,n+k)\gcd(d, n + k)gcd(d,n+k) 个大小为 dgcd⁡(d,n+k)\frac d{\gcd(d, n + k)}gcd(d,n+k)d​ 的环。

开 nnn 个完全背包，枚举 ddd，把 gcd⁡(d,n+k)\gcd(d, n + k)gcd(d,n+k) 加入到第 dgcd⁡(d,n+k)\frac d{\gcd(d, n + k)}gcd(d,n+k)d​ 个背包。最后假设有 cic_ici​ 个长度为 iii 的环，判断第 iii 个背包是否能凑出 cic_ici​ 即可。

时间复杂度：O(∑n2)\mathcal O(\sum n^2)O(∑n2)。

期望得分：404040。

算法 3

给可能存在的和 n+kn + kn+k 因数个数有关的做法，或者是加以优化的背包。

有一个做法是，因为 ci≤nic_i \le \frac nici​≤in​，我们对于第 iii 个背包可以只开到 ni\frac niin​。并且一个背包里最多只会有 d(n+k)d(n + k)d(n+k) 个数，于是我们有一个很松的上界是 O(nd(n+k)ln⁡n)\mathcal{O}(n d(n + k) \ln n)O(nd(n+k)lnn)。

期望得分：707070。

算法 4

注意到一个结论：若 xgcd⁡(x,n+k)=ygcd⁡(y,n+k)=g\frac x{\gcd(x, n + k)} = \frac y{\gcd(y, n + k)} = ggcd(x,n+k)x​=gcd(y,n+k)y​=g，那么令 z=gcd⁡(x,y)z = \gcd(x, y)z=gcd(x,y)，有 zgcd⁡(z,n+k)=g\frac z{\gcd(z, n + k)} = ggcd(z,n+k)z​=g。

证明是简单的，分别考虑每一个质因子 ppp，记 cx,cy,cz,c′cx, cy, cz, c'cx,cy,cz,c′ 分别表示这个质因子在 x,y,z,n+kx, y, z, n + kx,y,z,n+k 里的出现次数，那么有 cx−min⁡(cx,c′)=cy−min⁡(cy,c′)cx - \min(cx, c') = cy - \min(cy, c')cx−min(cx,c′)=cy−min(cy,c′)。这可以导出 cx=cycx = cycx=cy 或 cx,cy≤c′cx, cy \le c'cx,cy≤c′。令 cz=min⁡(cx,cy)cz = \min(cx, cy)cz=min(cx,cy)，两种情况都能进一步推出 cx−min⁡(cx,c′)=cy−min⁡(cy,c′)=cz−min⁡(cz,c′)cx - \min(cx, c') = cy - \min(cy, c') = cz - \min(cz, c')cx−min(cx,c′)=cy−min(cy,c′)=cz−min(cz,c′)。

那么从该结论可以推出上述的每个背包中的所有数都是最小数的倍数，否则选出一个不是最小数倍数的数，它与最小数的 gcd⁡\gcdgcd 更小。所以我们只保留背包中最小的数即可。

由于 gcd⁡(i,n+k)\gcd(i, n + k)gcd(i,n+k) 是关于 iii 的积性函数，可以做到线性。直接暴力求 gcd⁡\gcdgcd 可能也能过。

时间复杂度：O(∑n)\mathcal O(\sum n)O(∑n) 或 O(∑nlog⁡n)\mathcal O(\sum n \log n)O(∑nlogn)。

期望得分：100100100。
